# Rafael Pérez — Developer Portfolio

Personal portfolio built with **Next.js 14** and TypeScript. Fast, minimal, and designed to get straight to the point.

**Live:** [rafatrik.com](https://rafatrik.com)

---

## What's inside

- Animated hero with a typewriter subtitle loop
- Sticky nav with smooth scroll and active-section highlighting
- Projects section with live/GitHub links and status badges
- Scrolling tech stack ticker
- Client web work showcase
- Stats counter section
- Contact & socials

## Tech

- **Next.js 14** (App Router) + **TypeScript**
- Pure CSS — no UI library, no Tailwind
- `IntersectionObserver` for scroll-triggered reveal animations
- Zero runtime dependencies beyond React and Next

## Run locally

```bash
npm install
npm run dev
# → http://localhost:3000
```

## Deploy

Optimised for **Vercel** — just connect the repo and it deploys automatically on every push to `main`.
